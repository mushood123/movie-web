# 🎬 Movie App

Welcome to **Movie App** – the ultimate single-page web application for movie enthusiasts! Built with **React.js, Vite.js, and Tailwind CSS**, this app allows you to search for movies from **TMDb API** with a seamless **debounced search** experience. 🍿✨

---

## 🚀 Features
- 🔎 **Search movies in real-time** using debounced API calls.
- 🎨 **Sleek & modern UI** powered by Tailwind CSS.
- ⚡ **Superfast performance** with Vite.js.
- 🎞️ **Fetch movies dynamically** from TMDb API.
- 💻 **Fully responsive** for an immersive experience across devices.

---

## 🛠️ Tech Stack
- **Frontend:** React.js, Vite.js, Tailwind CSS
- **API:** The Movie Database (TMDb)
- **Package Manager:** npm

---

## 📦 Installation & Setup
Get started in just a few steps! 🎬💻

```sh
# Clone the repository
git clone git@github.com:mushood123/movie-web.git

# Navigate to the project directory
cd movie-app

# Install dependencies
npm install

# Run the development server
npm run dev

# Run locally on your laptop or router
npm run dev-host
```
